********************************************************************
This is a java application and requires Java version 1.5.
Try running the application as described below, and if
you encounter errors when running, make sure you are running
java version 1.5. You can test this by typing "java -version"
at the command prompt.
********************************************************************

To run, simply download the bioValidator.jar file and click on it in your
Finder or Explorer window.

If that doesn't work, then open up a command-prompt, navigate to the directory
where you downloaded the file and type the following:

java -jar bioValidator.jar
